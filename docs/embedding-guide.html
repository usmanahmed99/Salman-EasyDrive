Booking Links & Embedding Guide

How to deep-link and embed the online booking widget on easydriving.ca using URL query strings.
Booking origin: https://booking.easydriving.ca/book  ·  Updated June 2026

1. The basics

The public booking page lives at /book on the booking domain. Its starting state is controlled entirely through URL query-string parameters — no code changes are needed to point a button or link at a specific location, service, package, language, or tab.

Parameters can be combined freely with &. Anything you don't set falls back to a sensible default (English, the location picker, the Services tab).

2. Parameter reference

ParameterValuesWhat it doesDefault
center a location slug
laval, kirkland, henri-bourassa		 Preselects the location and opens directly on the service/package picker. Show the location picker
service a service slug
e.g. driving-lesson		 Preselects a single service and jumps straight to choosing a date & time. —
package new a package slug
e.g. road-test-prep		 Opens directly in the multi-session package booking flow. Takes precedence over service if both are present. —
tab new packages Opens the picker on the Packages tab instead of Services. (The tab bar only appears when the chosen location actually offers packages.) services
lang fr  /  en Sets the interface language. en
embed 1 Hides the page header/site chrome — use this when loading the page inside an iframe. Full page
Slugs. The exact center, service and package slugs come from the Admin Portal (each item's URL slug). The examples below use the seeded demo slugs; confirm production slugs before publishing links.

3. Scenarios

A · Link to the whole booking page
When: a generic "Book online" button. The customer picks location → service themselves.
<!-- starts on the location picker, English, Services tab -->
https://booking.easydriving.ca/book
B · Preselect a location
When: a page that is specific to one branch (e.g. the Laval location page).
/book?center=laval
C · Jump straight to one service
When: a "Book a driving lesson" call-to-action. Goes right to date & time.
/book?center=laval&service=driving-lesson
D · Jump straight into a package new
When: promoting a bundle such as the Road Test Prep package. Opens the multi-session flow.
/book?center=laval&package=road-test-prep
E · Open the picker on the Packages tab new
When: you want customers to browse packages first, but still choose which one.
<!-- location preselected, picker opens on the Packages tab -->
/book?center=kirkland&tab=packages
F · French deep link
When: linking from a French page. Add lang=fr to any of the above.
/book?center=laval&service=road-test-package&lang=fr
/book?center=laval&package=road-test-prep&lang=fr&tab=packages

5. Embedding with an iframe

To embed the widget inside a page on easydriving.ca, add embed=1 (hides the header) and combine it with any of the parameters above. The Pages content-security policy already allows framing by https://easydriving.ca, https://www.easydriving.ca, and the booking origin itself.

Example — embed the Packages tab for one location

<iframe
  src="https://booking.easydriving.ca/book?center=laval&tab=packages&embed=1"
  title="Easy Driving booking"
  style="width:100%;min-height:850px;border:0"
  loading="lazy">
</iframe>

Example — embed a single service, in French

<iframe
  src="https://booking.easydriving.ca/book?center=kirkland&service=car-rental&lang=fr&embed=1"
  title="Réservation Easy Driving"
  style="width:100%;min-height:850px;border:0"
  loading="lazy">
</iframe>
Before launch: test the iframe height on mobile, keyboard behaviour on form fields, and scrolling. Increase min-height if the package flow (which is taller) gets clipped.

6. Precedence & behaviour notes

  • package wins over service. If both are supplied, the package flow opens.
  • tab=packages is only honoured when packages exist for the selected location; otherwise the page shows Services (the tab bar is hidden).
  • Switching tabs updates the URL in the browser, so the customer can copy/share the link they're looking at.
  • Defaults: no lang → English; no tab → Services; no center → the location picker.
  • Order doesn't matter — ?service=x&center=y is the same as ?center=y&service=x.
